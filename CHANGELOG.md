# Changelog

## 1.0.0+1 (2026-02-26)

### 출시
- GlucoInsight 첫 출시 버전
- 성인 대상 당뇨 위험도 참고 예측 기능 제공 (심플/상세 모드)
- 나이, 성별, BMI, 허리둘레, 공복혈당(선택) 기반 예측 흐름 제공
- 주소 저장 및 주변 병원 검색/길찾기 연동
- 당뇨 건강정보(평가 참고 기준, 생활수칙, 응급 상황 안내) 제공
- 인앱 리뷰 요청 및 스토어 평점 이동 기능 적용
- 라이트/다크 테마 지원
